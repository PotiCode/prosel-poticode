## Como funciona o Processo Trainee

O Processo Trainee PotiCode é o programa de formação interna da empresa, desenhado para integrar novos membros e desenvolver habilidades em desenvolvimento web do zero, preparando-os para atuar nos projetos reais da PotiCode.

### O que você vai aprender

- Estruturação de páginas com HTML do zero
- Estilização e layout com CSS
- Lógica e interatividade com JavaScript
- Estilização moderna e responsiva com Tailwind CSS
- Desenvolvimento de sites com o framework Astro

### Estrutura das semanas

- **Semana 1** — HTML
- **Semana 2** — CSS
- **Semana 3** — JavaScript
- **Semana 4** — Tailwind CSS
- **Semana 5** — Astro

### Como acompanhar o progresso

Todos os materiais de estudo estão listados na seção de Capacitações, organizados por semana. Os conteúdos são disponibilizados progressivamente ao longo do programa.
